🌐 Html Serializer & Query Engine
הכלי שהופך דפי אינטרנט למבנה נתונים חכם
ברוכים הבאים לפרויקט ה-Html Serializer! הפרויקט הזה נולד כדי לגשר על הפער שבין קוד HTML גולמי (טקסט ארוך ומסורבל) לבין עולם האובייקטים של #C. בעזרת הכלי הזה, ניתן לנתח אתרי אינטרנט, לשלוף מהם מידע ולבצע שאילתות מורכבות בקלות.

✨ מה הפרויקט מציע?
🔍 ניתוח HTML חכם: המערכת יודעת לקרוא קוד HTML מהרשת, לזהות תגיות, מאפיינים (Attributes) וטקסט פנימי, ולבנות מהם "עץ משפחה" היררכי.

⚡ מנוע שאילתות עוצמתי: ממש כמו שמשתמשים ב-CSS כדי לעצב אלמנטים, הכלי מאפשר לחפש אלמנטים ספציפיים בעץ לפי שם התגית, ה-ID שלה או ה-Classes שלה.

🚀 ביצועים ואופטימיזציה: הפרויקט עושה שימוש במבני נתונים מתקדמים כדי להבטיח חיפוש מהיר, מניעת כפילויות בתוצאות ושמירה על יציבות הזיכרון גם באתרים עמוסים.

🧩 תשתית ל-Crawler: זהו צעד ראשון ובסיסי בבניית סורק אתרים (כמו של גוגל), המאפשר אינדוקס ושליפת נתונים אוטומטית.

🛠️ איך זה עובד מאחורי הקלעים?
שלב הטעינה: המערכת פונה לכתובת URL ומקבלת את מחרוזת ה-HTML.

שלב הפירוק: בעזרת ביטויים רגולריים (Regex), הטקסט מפורק ל"טוקנים" – חלקי תגיות וטקסט.

בניית העץ: המערכת עוברת על החלקים ובונת אובייקטים המקושרים זה לזה בקשרי "אבא-בן".

שאילתות (Querying): המשתמש מגדיר "סלקטור" (כמו div.main), והמערכת סורקת את העץ ומחזירה את כל האלמנטים המתאימים.

💡 למה להשתמש בזה?
למידה מעמיקה: הבנה של איך דפדפנים ומנועי חיפוש מנתחים את הרשת.

גמישות: ניתן להתאים את הכלי לכל סוג של אתר או צורך עסקי (השוואת מחירים, ניתוח טכנולוגיות וכו').

קוד נקי: עבודה עם אובייקטים מסודרים במקום חיפוש ידני בתוך מחרוזות טקסט.

⚙️ דרישות מערכת
סביבת .NET 6.0 ומעלה.

קבצי הגדרות (JSON) המכילים את חוקי ה-HTML.
