✅ ToDo List - Fullstack App
אפליקציה מאובטחת לניהול משימות בטכנולוגיית Minimal API ו-React
פרויקט זה מציג פתרון Fullstack מקצה לקצה לניהול רשימת משימות. האפליקציה נבנתה תוך שימוש בטכנולוגיות החדישות ביותר של מיקרוסופט (Minimal API) בשילוב עם React, תוך דגש על אבטחת מידע באמצעות JWT, ביצועים גבוהים וחוויית משתמש חלקה.

🏗️ ארכיטקטורת המערכת
המערכת בנויה משלושה נדבכים המשלימים זה את זה:

Backend (Server): מבוסס .NET Minimal API. ארכיטקטורה רזה המאפשרת ביצועים מקסימליים וקוד נקי ומרוכז.

Database: מסד נתונים MySQL. העבודה מבוצעת בגישת DB-First עם Entity Framework Core, המבטיחה סנכרון מושלם בין בסיס הנתונים לקוד ה-C#.

Frontend (Client): ממשק משתמש דינמי ב-React, המנהל תקשורת מאובטחת מול השרת ומספק חווית ניהול משימות אינטואיטיבית.

🔐 אבטחה והזדהות (JWT)
אבטחת המידע עומדת בליבת הפרויקט:

Authentication: מערכת הרשמה והתחברות מלאה המבוססת על JWT (JSON Web Token).

Protected Routes: ה-API מוגן כך שרק משתמשים מאומתים בעלי טוקן תקף יכולים לגשת למשימות שלהם.

Axios Interceptors: מימוש Interceptor בצד הלקוח המזהה שגיאות הרשאה (401) ומנתב את המשתמש אוטומטית לדף ההתחברות.

🚀 תכונות מרכזיות (Features)
ניהול משימות מלא (CRUD): יצירה, שליפה, עדכון סטטוס ומחיקה של משימות בזמן אמת.

API תכליתי: שימוש ב-Minimal API לניהול נתיבים (Routes) ישירות ב-Program.cs.

ממשק Swagger: תיעוד ובדיקה של כל פונקציות השרת בצורה ויזואלית ונוחה.

ניהול שגיאות חכם: Interceptor שתופס שגיאות Response ורושם אותן ללוג בצורה מסודרת.

CORS Policy: הגדרת הרשאות גישה המאפשרות עבודה בטוחה בין דומיינים שונים.

🛠️ טכנולוגיות בשימוש
Backend: .NET 6.0/7.0, Entity Framework Core, MySql Connector, JWT Auth.

Frontend: React, Axios, React Router.

Tools: VS Code, Dotnet CLI, MySQL Workbench.

⚙️ שלבי התקנה והרצה
בסיס הנתונים: צרו Schema ב-MySQL בשם ToDoDB והריצו את הסקריפטים ליצירת טבלאות המשימות והמשתמשים.

השרת (API): הגדירו את ה-Connection String ב-appsettings.json והריצו dotnet watch run.

הלקוח (React): התקינו את החבילות עם npm install והריצו את האפליקציה עם npm start.
